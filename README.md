# NestaPress

### Customisable, responsive blogging theme for the Nesta CMS.

Mockups are created in [Inkscape](http://inkscape.org) using the [SVG format](http://en.wikipedia.org/wiki/Scalable_Vector_Graphics). Nice and open.
